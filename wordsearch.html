    • O'odham plural/reduplicated forms (e.g., "hoaha")

  • Different O'odham words form plurals in different ways,these pattern numbers group words that pluralize similarly.
  • See the section below for detailed explanations of each pattern.


O'odham Reduplication Patterns Guide

In O'odham, most nouns form their plural by reduplication – copying part of the word and inserting it back into the word. There are six main patterns to learn, a seventh category for irregular forms that don't follow predictable patterns and one special rule for words that begin with w or v.


The Seven Reduplication Patterns

Pattern Rule / Description Example(s)
Pattern 1: Copy C Copy the first consonant (C) and insert it after the first vowel gaso → ga-g-so (fox → foxes)
Pattern 2: Copy CV Copy the first consonant AND vowel (CV) and insert after the first vowel novĭ → no-no-vĭ (hand → hands)
Pattern 3: Copy C and Shorten First Vowel Copy the first consonant (C), insert it, and shorten the first vowel (a: → a) pa:do → pa-p-do (duck → ducks)
Pattern 4: Copy CV and Shorten First Vowel Copy the first consonant and vowel (CV), insert them, and shorten the first vowel (i: → i) si:s → si-si-s (sibling → siblings)
Pattern 5: Copy CV and Lengthen First Vowel Copy the first consonant and vowel (CV), insert them, and lengthen the first vowel (a → a:) ban → ba:-ba-n (coyote → coyotes)
Pattern 6: Glottal Stop and Copy VC Insert a glottal stop (') after the first vowel, then copy the first vowel and consonant (VC) and insert after the glottal stop sikol → si-'is-kol (circle → circles)
Pattern 7: Irregular Plurals These plurals cannot be predicted, they have to be memorized individually viappoi → vi:piop (boy → boys)
bahi → ba:bhai (tail → tails)
'ali → 'a'al (baby → babies)

Special Rule: Words Beginning with W or V

When a word begins with w or v, follow these steps:

  1. Change the w or v to p
  2. Make any other changes as usual

Examples:

Singular Plural Pattern Meaning
vakial vapkial Pattern 1 cowboy → cowboys
vaika vapaika Pattern 2 ditch → ditches
va:so vapso Pattern 3 can → cans
vo:g vopog Pattern 3 road → roads
vag va:pag Pattern 5 hole → holes
vadadk va'apdadk Pattern 6 shiny/bald → (pl.) shiny/bald

Key pattern: w/v → p + copy vowel

Quick Reference Chart

Pattern What to Copy Vowel Change Example

2 CV no change novĭ → nonovĭ

Reduplication Tips

  1. Identify the first consonant and vowel in the singular form
  2. Determine which pattern the word follows (this requires memorization)
  3. Apply the copying rule for that pattern
  4. Check vowel length – does it stay the same, shorten, or lengthen?
  5. Remember the w/v rule – these consonants always change to p!
